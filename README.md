# DroidPad for Android

This is a PWA-ready Notepad application optimized for mobile devices.

## How to Build for Android

1. Install Capacitor: `npm install @capacitor/core @capacitor/cli`.
2. Initialize Capacitor: `npx cap init`.
3. Add Android platform: `npx cap add android`.
4. Copy these files into the `www` directory.
5. Run `npx cap copy` and `npx cap open android`.

## PWA Usage
Simply open the hosted `index.html` in Chrome on Android and select "Add to Home Screen" from the menu.
